## Tecnologias Utilizadas

- Backend: Spring Security, Spring Boot, Spring Data JPA e Thymeleaf  
- Frontend: JavaScript, CSS  
- Banco de Dados: **MySql**  
- Build e Deployment: Maven  
- Controle de versão: Git (GitHub)
